Dobrodošli na Ignitech Školica by Hrvoje Pavić

Ovo je početna stranica, riješenja zadataka su grupirana smisleno, da nije sve na jednom mjestu.
Tijek misli i dijela kojim je kreirana ova aplikacija je sljedeći :

1. Novi .NET Core Web API projekt sa potrebnim Paketima za Entity Framework

2. Kreirani modeli i AppDbContext (veza sa bazom)

3. Kreirana početna migracija sa Add-Migration InitialCreate, Update Database

4. Dodani Seed podaci

5. Projekt spojen na novi Git repozitorij (link), dodan .gitignore

6. Back metode dovršene: kreirana 4 kontrolera, 1 servis, 2 helper metode

7. Instaliran Front Angular projekt, pokretanje back i front u isto vrijeme preko Concurrently sa npm start iz foldera IgnitechSkolicaFrontend

8. Back Swagger na : http://localhost:5090/swagger, front na : http://localhost:4200/

9. Kreirani servisi za dohvat podataka sa back-a i komponente za prikaz podataka u frontu

10. Dodan final touch CSS
